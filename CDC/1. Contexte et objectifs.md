De nombreuses associations (SPA, refuges, fondations animales) ont besoin d’un système centralisé pour gérer les animaux accueillis, les adoptions, les bénévoles, et les opérations de sauvetage.

Pour ce faire, ils vous sollicitent, vous, **Nils, futur expert CDA Handigital !!**

  
Le but est de créer une **application web sécurisée**, permettant de suivre le parcours de chaque animal, d’automatiser la prise de rendez-vous d’adoption et de gérer les dossiers administratifs.

Le projet est aligné avec :

- les obligations **RGPD** (dossiers adoptants)
    
- l’accessibilité **RGAA**
    
- les bonnes pratiques de sécurité **ANSSI**
    

L’application doit offrir un volet public et une interface d’administration complète.
