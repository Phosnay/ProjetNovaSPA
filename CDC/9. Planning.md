##  JANVIER – Phase 1 : Analyse + Maquettage + Conception

**Objectif :** base du projet bien définie + dossier de conception complet.

**Livrables :**

- Analyse des besoins (user stories, use cases)
- Zoning + wireframes
- Maquettes Figma complètes (desktop + mobile)
- Schéma d’enchaînement des écrans
- Dossier de conception (architecture + technologies + sécurité)
- MCD + MLD + MPD
- Choix des outils / installation environnement / dépôt Git
- Planning projet (Gantt ou Scrum backlog)

_Compétences couvertes du CDA : CP5, CP6, CP7, CP4_

##  FÉVRIER – Phase 2 : Développement UI (Frontend)

**Objectif :** interfaces statiques puis dynamiques.

**Livrables :**

- Structure HTML / CSS complète
- Composants UI interactifs
- Adaptations responsive
- Début intégration API mockée
- Tests unitaires simples des composants
- Documentation code

_Compétences couvertes du CDA : CP2_

##  MARS – Phase 3 : Backend + Base de données

**Objectif :** commencer le fonctionnement réel.

**Livrables :**

- API REST sécurisée
- CRUD complet sur les entités principales
- Authentification / rôles
- Connexion base de données
- Services métiers opérationnels
- Sécurisation des entrées / gestion erreurs
- Tests unitaires backend
- Jeu d’essai en base

_Compétences couvertes du CDA : CP3, CP7, CP8_

##  AVRIL – Phase 4 : Intégration complète + Tests

**Objectif :** frontend ↔ backend + fiabilisation.

**Livrables :**

- Connexion complète UI ↔ API
- Accessibilité vérifiée (RGAA niveau AA simplifié)
- Tests d’intégration
- Tests de sécurité (injection, auth, XSS…)
- Tests utilisateur (acceptation)

_Compétences couvertes du CDA : CP9_

##  MAI – Phase 5 : Déploiement + Documentation + DevOps

**Objectif :** l’application doit tourner en ligne.

**Livrables :**

- Dockerisation de l’app
- Déploiement sur serveur cloud
- CI/CD opérationnel (build + tests automatiques)
- Scripts de déploiement
- Procédure d’installation documentée
- Documentation utilisateur
- Documentation technique complète

_Compétences couvertes du CDA : CP10, CP11_

**Objectif :** stabilisation + préparation au passage du titre.

**Livrables :**
